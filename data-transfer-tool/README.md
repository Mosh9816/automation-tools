# データ転記ツール（Excel VBA）

本ツールは、日次で更新される元データを安全に転記し、  
件数チェックを行ったうえで保存する **日次処理自動化ツール** です。

Excel 上の **「クリックで転記開始」ボタン** を押すだけで、  
以下の処理が自動で実行されます。

---

## ■ 動作内容（処理フロー）

1. **更新日チェック（鮮度確認）**  
   - 元データの更新日を取得し、当日更新されているか確認  
   - 古いデータの誤転記を防止

2. **データ転記（A列コピー）**  
   - 元データの A 列を最終行まで自動取得  
   - 新規ブックに値貼り付けで転記

3. **件数チェック（整合性確認）**  
   - 転記元と転記先の件数を比較  
   - 抽出漏れ・転記漏れを検出

4. **保存処理（フォルダ自動作成）**  
   - 保存先フォルダが無ければ自動作成  
   - `保存用データ_yyyymmdd.xlsx` の形式で保存  
   - 日次処理の履歴管理を実現

---

## ■ 保存先（※各自のPC環境に合わせて書き換えが必要）

本ツールでは、以下のフォルダに保存する仕様になっています：

```
C:\Users\YourName\Desktop\日時処理データ\
```

※ **YourName の部分は各PCのユーザー名に置き換えてください。**  
例：

```
C:\Users\shoko\Desktop\日時処理データ\
```

保存されるファイル名：

```
保存用データ_20260428.xlsx
```

---

## ■ 想定している元データ（ダミー）

本ツールで使用する元データは、以下の2列のみを想定しています。

```
A列：TEL（ダミー番号）
B列：顧客名（ダミー）
```

実務で使用していた構造を、個人情報を含まない形で再現しています。

---

## ■ UI（画面イメージ）

- 大きな「クリックで転記開始」ボタン  
- 下部に処理内容（更新日チェック → 転記 → 件数チェック → 保存）を表示  
- 初めて使う人でも迷わないシンプル設計

---

## ■ 使用技術

- Excel VBA  
- FileDateTime による更新日チェック  
- 最終行自動取得  
- CountA による件数チェック  
- MkDir によるフォルダ自動作成  
- Workbooks.Add による新規ブック生成

---

## ■ 想定利用シーン

- 日次で更新されるデータの転記作業  
- 履歴を残しながらのデータ管理  
- 転記ミス防止・品質担保が必要な業務
