# automation-tools
  データ事務などの実務経験をベースにAIを活用して作成した自動化ツールです。  
---

## 📦 ツール一覧  
日々の事務作業を効率化するために実務で使用していた処理をベースに作成しています。  

- [Data Transfer Tool](./data-transfer-tool)  
  Excel 間のデータ転記を自動化する業務効率化ツール。
  
- [CSV Split Tool](./csv-split-tool)  
  Excel のブロック構造データを解析し、項目番号ごとに CSV を自動生成するツール。

- [Format Generator](./format-generator)  
  日次チェック作業のフォーマットを自動生成する Excel VBA ツール。

- [data-check-tool](./data-check-tool)  
  取込データと system_data を照合し、OK/NG を自動判定するチェックツール。

- [mail-auto-tool](./mail-auto-tool)  
  NG 行をもとに Outlook 下書きを自動生成するツール。

- [pdf-rename-tool](./pdf-rename-tool)  
  system_data の管理番号・文書名に基づき、PDF を一括で命名規則に沿ってリネームするツール。

- [pdf-rename-runner](./pdf-rename-runner)  
  ZIP ダウンロード → 解凍 → PDF リネーム → バックアップまでを自動化する PAD フロー。
