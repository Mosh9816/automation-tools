# PDF Rename Runner（PAD）

## 📝 概要
Excel の PDF Rename Tool を PAD から実行し、
フォルダ選択 → マクロ実行 → 完了通知までを自動化するフロー。

## 🎯 目的
- Excel マクロの実行をワンクリック化
- 手動操作の削減
- RPA と VBA の連携を実務的に再現

## ⚙️ フロー構成
1. Excel 起動
2. マクロ実行
3. 完了メッセージ表示

flow.json  
assets/（スクリーンショット）
